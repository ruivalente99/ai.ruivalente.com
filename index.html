03 · TOKEN ECONOMY
// 03, tokens are the new electricity bill

Every word costs.

Please , can you kindly help me summar ize this long document ? → 13 tokens
// politeness is invisible to the model, and visible on the bill.
✕ Wasteful
  • ×"Hello! Thank you!"  pleasantries, padding, sign-offs
  • ×"Please, could you maybe..."  hedging, repetition
  • ×Dumping the whole chat history into every call
  • ×Re-pasting the same long doc, every request
  • ×Vague asks that force back-and-forth clarification
✓ Token-aware
  • ✓Direct verbs.  "Summarize." "Extract." "Classify."
  • ✓Structure beats prose.  Headers, lists, delimiters
  • ✓Clean context window.  only what matters now
  • ✓Cache stable parts.  pay once, reuse forever
  • ✓Specify the output format up front

Token Context Window Prompt Caching Input vs. Output Cost

  • Autonomous handoff. Work passes between them without a human in every step.

~/career/glossary
05 · INTERVIEW VOCABULARY
// 05, three words that signal "in the room"

Speak it precisely.

→ TERM 01
Skills
// packaged capabilities
Reusable competencies you give a model, a defined function it can call on demand.
like → superpowers on a resume, but for an AI.
→ TERM 02
Connectors
// bridges to real systems
The wiring that lets an AI reach outside its chat box, into calendars, databases, tickets, code.
like → giving the model hands and a phone.
→ TERM 03
MCP
// Model Context Protocol
An open standard for how AI tools talk to external data & software, consistently, securely, anywhere.
like → USB-C, but for AI integrations.
+ Bonus terms → Context Window · working memory Grounding · anchored in real sources Eval · how you prove it works
memorize these.
